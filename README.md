# Vocab App

An adaptive English vocabulary learning app. Every word served to the user is chosen based on their **field of interest** (Engineering, Health, Law, Sports, Education) and their current **English level** (A1–C1). The core differentiator is field-aware curation + celebrity voice AI feedback.

## Tech Stack

- **Mobile Environment**: React Native + Expo (SDK 52+)
- **Navigation**: Expo Router
- **Backend / Database / Auth**: Appwrite
- **State Management**: Zustand
- **TTS / Voice Integration**: ElevenLabs API
- **AI (Word filtration & explanations)**: Claude API
- **Animations / Gestures**: react-native-reanimated + react-native-gesture-handler

## Key Features

- **Adaptive Learning**: Words are tailored to the user's selected field and English proficiency level.
- **Spaced Repetition**: Intelligent review intervals to ensure vocabulary mastery.
- **Celebrity Voice Feedback**: Engaging audio pronunciations and context sentences.
- **Guest Mode**: Try the app with a limited daily vocabulary without needing an account.
